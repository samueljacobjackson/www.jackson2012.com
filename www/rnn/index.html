A PC's Comedy Routine

-or-

Why Computers Won't Steal Any Comedian's Job.

This is a comedy routine written by a computer using recurrent neural networks. I used Andrej Karpathy's implementation of torch-rnn. I have input comedy routines from some of the biggest names of the decade. The output is an strange jumble of jokes that don't make sense. There are a few funny ones though.

The raw text can be found here.
