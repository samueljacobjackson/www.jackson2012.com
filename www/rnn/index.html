A Comedy Routine

Or why computer's won't steal any comedian's job.

This is a comedy routine built with recurrent nuural networks using Andrej Karpathy's implementation of torch-rnn. I have input comedy routines from some of the biggest names of the decade. And the output is an strange jumble of jokes that don't make sense. There are a few funny ones though.
