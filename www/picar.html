Raspi Car

This is my project to make an internet controlled car from an old RC car and a Raspberry Pi. It consists of a web server running on a Raspberry Pi Zero W, a webcam and an old RC car. The components may be purchased for under $45.00 making this an affordable and fun project. Some soldering is required so you will need a soldering iron and some jumper wires.

Overview

  A Raspberry Pi Zero W is a small single board computer made by the Raspbery Pi Foundation. For more information visit https://www.raspberrypi.org/ It is perfect becaus of it's small size and built in Wifi. The Pi will require a micri SD card to hold the Linux OS. An 8GB or even a 4GB card is sufficient for this project. Less than 4 is too small. Setting it up will require a mini HDMI cable and a TV in which to plug it. Once the initial set up is done there will no longer be a need for the cable.

  I recommend a Creative - Live! Cam which may be bought at Best Buy. You may be able to find a less expensive one on ebay, just avoid shipments from china. An OTG (On The Go) Micro USB-to-USB is needed to connect the camera to the Pi.

  The most difficult component to get is the RC Car. Newer cars in the stores use lipo batteries and have smaller electronics making them difficult to use. Thrift stores like Salvation Army or Goodwill are the best place. Having sid that the cars there are sort of hit or miss. Often times they may not work, but for a price of 4-6 dollars a couple of duds won't break the bank. If you have bought more duds than you wanted and are feeling ambitious you could hobble together a car from working parts. If you are unable to find one with a battery you will need to purchase one which will increase the cost. WHen buying a car note the voltage required. Some cars use standard double A batteries and only run on 6 volts. Those will be difficult to use. The best cars will use a 9.6 volt battery.

Other items required include:
  • Jumper wires. Male to female prefered.
  • 24 or 26 gage wire.
  • A USB adapter for a car to convert the voltage to 5 volts.
  • Tamiya Big Style Connector Plugs Unwired.
Tools needed:
  • Soldering iron.
  • Phillips screwdriver.
  • Wire strippers.
Useful tools:
  • Multimeter.

Install Raspian

  Download Raspiban from https://www.raspberrypi.org/downloads/raspbian/ Get the lite version because there will no need for a user interface. Next download Win32 Disk Imager Here Run the disk imager and select the .img file and your SD Card to install the OS. After that is done it is time to boot the Pi. Connect it to a TV or Monitor and a keybord then provide power. After it boots login with the default user name "pi" and password "raspberry". Before anything we need to get the device onto the wifi. (For a more detailed guid visit Raspberrypi.org's guide

Edit /etc/wpa_supplicant/wpa_supplicant.conf to add wifi by typing: 

  sudo nano /etc/wpa_supplicant/wpa_supplicant.conf
Edit the file to have these lines. 

  network={
    ssid="name_of_wifi"
    psk="passkey_of_wifi"
  }

Press ctrl-X to save and exit.

By default Raspbian has SSH disabled. To enable it type these lines. 

  sudo systemctl enable ssh 
  sudo systemctl start ssh

  Reboot by typing sudo reboot Once it has rebooted login and type ifconfig and note the IP address. The Pi may now be disconnected from the TV/Monitor and keyboard; they are no longer needed. Further interfacing with the Pi can be dome remotely with a SSH client. Putty, WinSCP, or MobaXterm will work. Select your favorite and log into the IP address from before with the defaul username and password.
